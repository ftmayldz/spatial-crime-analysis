# Spatial Crime Analysis

This project analyzes crime data from Chicago to identify patterns in crime distribution across time and location.

## Objectives
- Analyze crime trends by hour and month
- Identify the most common crime types
- Visualize crime locations using maps
- Detect high crime zones using clustering

## Methods
- Data Cleaning with Pandas
- Exploratory Data Analysis
- Spatial Visualization using Folium
- Heatmap visualization
- KMeans clustering for crime zones
- Basic machine learning model

## Dataset
Chicago Crime Dataset (2001–Present)
